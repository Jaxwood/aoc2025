# Advent of Code 2025 [![Tests](https://github.com/Jaxwood/aoc2025/actions/workflows/tests.yml/badge.svg)](https://github.com/Jaxwood/aoc2025/actions/workflows/tests.yml)

My solutions for [Advent of Code 2025](https://adventofcode.com/2025), implemented in Python.

## Prerequisites

- [mise](https://mise.jdx.dev/) (manages Python and uv versions)

## Installation

### Install mise (if not already installed)

```bash
# On macOS and Linux
curl https://mise.run | sh

# Or with Homebrew
brew install mise
```

### Set up the project

```bash
make install
make sync
```

## Development

### Running Tests

```bash
# Run all tests
make test

# Run tests with verbose output
make test-v
```
